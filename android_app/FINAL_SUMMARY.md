**Q: How long does the first build take?**
A: 20-30 minutes depending on your internet and computer speed. Subsequent builds are faster (5-10 min).

**Q: Do I need Docker installed?**
A: Only if using build.bat on Windows. You can use Linux/Mac local build or online builder instead.

**Q: Can I test without a physical device?**
A: Yes, use Android emulator in Android Studio, but test on real device before release.

**Q: What if the app won't connect to backend?**
A: Check:
1. Backend URL is correct in main.py
2. Flask server is running
3. Phone is on same network
4. Firewall allows port 5000

**Q: Can I publish to Google Play Store?**
A: Yes! Sign the APK with production keystore and upload to Google Play Console ($25 one-time fee).
